### US-002: Build setup flow for players and situation selection
Status: in_progress
Depends on: US-001

Description:
As a host, I want to enter player names and choose a specific or random situation so I can start a valid round quickly.

Acceptance Criteria:
- [ ] Provide input rows for player names with add/remove controls and enforce minimum 4 players.
- [ ] Provide a situation selector with a Random option that chooses only compatible situations.
- [ ] Display setup status including ready player count, minimum required, and max supported for selected situation.
- [ ] Example: entering 4 unique names and choosing Random enables starting a round.
- [ ] Negative case: attempting to start with duplicate names like 'Alex' and 'alex' shows a validation error and blocks start.
